## üìÇ Project Overview

This repository serves as the **only authentic and most up-to-date** collection of **unaltered stock OTA images**, **OTA update links**, and **full firmware packages** for [**Nothing**](https://nothing.tech) and [**CMF by Nothing**](https://cmf.tech) devices available on the open web. This is achieved by [**indexing**](#-supported-models) both incremental and full firmware package URLs (when available) directly from official update servers, and [**archiving**](https://archive.org/details/nothing-archive) them accordingly. Additionally, [**stock OTA images**](https://github.com/spike0en/nothing_archive/releases) are generated by applying incremental updates over the base full firmware package using AOSP's OTA extraction tool. This project assists with flashing, sideloading, and preserving official firmware for various scenarios where it may be applicable.

## üìå Use Cases  

This archive is a reliable solution for a variety of scenarios involving **Nothing** and **CMF** devices:
- **Sideloading:**  Manually install updates for devices when region-based rollouts are staggered or when the device is not receiving the OTA for various reasons.
- **Rooting & Unrooting:** Access stock boot images for rooting your device using available tools, and safely unroot it when necessary to install official OEM updates.
- **Restoring or Flashing Stock Firmware:** Revert to stock firmware after custom ROM installations or prepare your device for custom ROMs.
- **Firmware Archiving:** Archive and preserve the oem firmware versions for future use or testing purposes.
